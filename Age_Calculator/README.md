# Age Calculator using Python

## Description

A command-line Age Calculator built using Python that calculates a person's exact age based on their date of birth.

The program displays:

- Exact Age (Years, Months, Days)
- Total Completed Months
- Total Days Lived

It also validates invalid dates and prevents future date entries.

---

## Features

- Calculate exact age
- Calculate total completed months
- Calculate total days lived
- Input validation
- Future date validation
- Clean CLI interface
- Modular function-based design
